## ✅ 已完成功能（相比 v1.0.0-stable）

### 基础功能（继承自 v1.0.0-stable）

| 功能 | 描述 |
|------|------|
| **文本对话** | 完整聊天界面，实时消息收发，消息历史显示 |
| **语音对话 · 录音** | Android 原生 AudioRecord，48kHz 采集，VAD 检测 |
| **语音对话 · 播放** | PCM 音频流实时播放，48kHz AudioTrack |
| **实时重采样** | 48kHz → 16kHz 自动重采样后上传 |
| **WebSocket 通信** | 与后端双向实时通信，断线重连，JSON/Binary 混合帧 |
| **Live2D 角色渲染** | Cubism SDK 集成，角色模型显示与动画 |
| **唇同步（LipSync）** | 音频振幅实时驱动角色口型，低延迟 |
| **QR 扫码连接** | 扫 nekorn:// deep link，自动写入后端配置并跳主界面 |
| **麦克风权限管理** | 友好弹窗、拒绝后引导跳设置、延迟请求避免冲突 |
| **音频诊断工具** | /audio-debug 页面：采样率检测、错误报告 |
| **跨平台音频架构** | @project_neko/audio-service，Native/Web 双实现 |
| **原生 PCM 模块** | react-native-pcm-stream，Kotlin 实现 |
